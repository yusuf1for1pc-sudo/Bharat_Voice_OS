# Bharat Voice OS — Voice-Driven Android Automation Platform

Bharat Voice OS is a voice-first AI system that enables users to operate Android smartphones entirely through natural speech in Indian languages.
It performs real in-app tasks automatically - no reading, typing, or menu navigation required.
Built to deliver digital independence for non-literate smartphone users.

🔗 Demo / Access
Repo: [Github repo link](https://github.com/yusuf1for1pc-sudo/health-surveillance-system.git)


🚀 What the System Does

-  Users speak naturally in any of 11 supported Indian languages
-  The system understands intent in real time
-  It opens and navigates Android apps automatically
-  Completes tasks end-to-end
-  Confirms results through voice output
  
🎯 Core Use Cases

-  🚕 Book a cab using voice
-  🏛 Check government scheme status
- 🌾 Agricultural advisory via voice queries

👥 Target Users

- Semi-literate and non-literate smartphone users
- Rural and Tier-2/Tier-3 city populations
- Users dependent on middlemen for digital tasks

## Tech Stack

- **Frontend**: Android Native (Kotlin) + Android Accessibility Service + MediaRecorder API (On-device audio capture)
- **Backend**: FastAPI (Python) + REST API (JSON orchestration)
- **AI**: Sarvam STT, Sarvam-M, Sarvam Bulbul, Gemini Vision API

## Features

- 🎤 Voice-to-Action Automation — Speak naturally in Hindi/Marathi and complete real in-app tasks
- 📱 Full Android Screen Control — AI navigates menus, fills forms, and taps buttons automatically
- 🧠 Intent Understanding 
- 👁 Vision-Based Agent Loop 
- 🔁 End-to-End Task Completion 
- 🗣 Natural Voice Confirmation 
- 🔐 Privacy-First Design
